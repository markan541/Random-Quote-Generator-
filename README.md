# Random-Quote-Generator-
Автор:Андриевских Марк

## Описание

GUI-приложение для генерации случайных цитат. Позволяет:
- Генерировать случайные цитаты из предопределённого списка
- Добавлять новые цитаты с проверкой на пустые поля
- Фильтровать цитаты по автору и теме
- Просматривать историю сгенерированных цитат с временными метками
- Автоматически сохранять все данные в JSON-файл

## Требования

- Python 3.6+
- Стандартные библиотеки: `tkinter`, `json`, `random`, `os`, `datetime`

## Установка и запуск

1. Скачайте файл `quote_generator.py`
2. Запустите команду:
   ```bash
   python quote_generator.py
